# Changelog

### Added
- Initial release
- `PeeplClient` as main entry point
- Resources: `Contacts`, `Organisations`, `AddressTypes`, `Groups`, `Functions`, `Combinations`, `Activities`
- `PeeplPaginatedResult` helper with `hasNextPage()` / `hasPreviousPage()`
- `PeeplException` with `getHttpStatus()` and `getResponseBody()`
- PHPUnit test suite
